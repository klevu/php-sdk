BaseUrlsProvider.php : 171

Updates properties based on information found in Account model

public updateFromAccount(AccountInterface $account) : void

Rather than instantiating a new provider after retrieving an AccountInterface object in order to use the applicable URLS, the existing instance can be updated with a passed account model. This provides better support for container / DI patterns reusing the same provider throughout a request.
