Whether custom (ie not immutable) attributes can use this datatype

public isAvailableToCustomAttributes() : bool

Klevu's indexing will automatically register a number of core attributes (which return with the immutable flag). These attributes may use data types which cannot be used when registering custom attributes (see AttributesServiceInterface::put()) Where an attribute which is not immutable uses a data type which is not available to custom attributes, validation will fail before the request is sent (see AttributeValidator::execute())
